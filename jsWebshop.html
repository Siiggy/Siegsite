Demo project: Online Webshop

Build with HTML, EJS, CSS, Javascript, Node JS & MongoDB

In een web development bootcamp cursus heb ik een online webshop gemaakt waarin alles samen kwam wat ik in de cursus heb geleerd.

De webshop heeft een inlog-functie, producten die in een MongoDB database gestored zijn, een shopping cart functie en een betalings API.
Daarnaast is er ook een admin gedeelte waarin de producten en orders bijgehouden kunnen worden.
Verder is het project gestructureerd volgens het MVC-model.

Hieronder staan enkele plaatjes van de online webshop om een indruk te krijgen van het hele project:

Customer: Home Page

Customer: Product Page

Customer: Shopping Cart

Customer: Payment Redirect

Admin: Manage Products page

Admin: Manage Orders page
