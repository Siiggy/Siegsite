Als eerst wordt de video geladen via het HTML document van deze pagina. 
Dit gebeurd via de volgende syntax:

<video autoplay loop muted>
	<source src="/Media/videos/Ristar-video.mp4">
</video>

In de volgende snippet zien we hoe deze via Sass CSS opgezet en gestyled wordt.



.videoContainer {
	height: 700px;
	width: 100%;
	position: relative;
	z-index: 1;

.filter {
	height: 700px;
	width: 100%;
	background: #B5189F;
	background: -webkit-linear-gradient(top left, #B5189F, #4461EE);
	background: -moz-linear-gradient(top left, #B5189F, #4461EE);
	background: linear-gradient(to bottom right, #B5189F, #4461EE);
	position: absolute;
	z-index: 2;
	opacity: 0.7;
}

video {
	position: absolute;
	object-fit: cover;
	height: 700px;
	width: 100%;
}
}
